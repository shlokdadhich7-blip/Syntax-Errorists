# KisanKredit 🚜🌾

**KisanKredit** is a premium, editorial-grade digital agri-lending platform designed to secure the future of Indian agriculture. It provides farmers with a high-precision interface for loan eligibility appraisal, digital capital requests, and sophisticated repayment tracking.

---

## ✨ Features

- **🎯 AI-Driven Eligibility Appraisal**: Utilizing the Google Gemini API to provide hardware-grade farm metrics appraisal and real-time loan eligibility insights.
- **📝 Elite Digital Questionnaire**: A streamlined, 4-phase digital application flow designed for farmers to apply for credit with minimal friction.
- **📊 Amortization Ledger (EMI Planner)**: A high-end FinTech dashboard to analyze repayment windows, track paid capital, and monitor pending risks.
- **🔄 Multi-Profile Overview**: A central hub to switch between different farmer profiles, allowing comparative analysis and managed oversight for cooperative societies or individual farmers with multiple assets.
- **🗣️ Bilingual Support (EN/HI)**: An inclusive, farmer-friendly interface with consistent Hindi translations for all critical actions and data points.
- **📖 Integrated User Guide**: A visual, step-by-step roadmap designed to help first-time users navigate the digital lending ecosystem effectively.
- **📈 Live Application Tracking**: A 5-stage verification pipeline to track loan status from submission to final disbursement.

## 🛠️ Technology Stack

- **Frontend**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) (Modern utility-first architecture)
- **Animations**: [Motion](https://motion.dev/) (High-frequency, non-linear UI transitions)
- **Backend/Server**: [Express](https://expressjs.com/) + [TSX](https://github.com/privatenumber/tsx)
- **Database/Auth**: [Firebase Firestore](https://firebase.google.com/)
- **AI Integration**: [Google GenAI (@google/genai)](https://ai.google.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- Personal Firebase Project
- Google Gemini API Key

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/kisankredit.git
   cd kisankredit
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env` file in the root directory and add your credentials:
   ```env
   GEMINI_API_KEY=your_gemini_api_key
   # Add your Firebase config if needed
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:3000`.
